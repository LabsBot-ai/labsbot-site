# LabsBot.ai — Project Definition / Описание проекта

---

## 1. Purpose / Назначение

**EN:**  
LabsBot.ai is a central web platform (hub) that presents and connects all LabsBot projects.  
This is NOT a landing page and NOT a marketing-only website.

The site must be suitable to:
- be shared publicly
- be linked from social networks
- represent a serious, long-term tech platform
- scale into a multi-product system

**RU:**  
LabsBot.ai — это центральная веб-платформа (хаб), объединяющая и представляющая все проекты LabsBot.  
Это НЕ лендинг и НЕ маркетинговый сайт.

Сайт должен:
- не стыдно показываться публично
- использоваться как ссылка в соцсетях
- выглядеть как серьёзная долгосрочная платформа
- масштабироваться в систему из нескольких продуктов

---

## 2. Target Audience / Аудитория

**EN:**  
Primary:
- Regular users
- Migrants
- Non-technical people
- Users who value simplicity

Secondary:
- Partners
- Investors
- Tech-savvy users

UI principle:  
Human-first, simple, friendly.  
No complex dashboards on the main site.  
No heavy B2B-style interfaces.

**RU:**  
Основная аудитория:
- обычные пользователи
- мигранты
- нетехнические люди
- люди, которым важна простота

Вторичная аудитория:
- партнёры
- инвесторы
- технические специалисты

Принцип интерфейса:  
Сначала человек, потом технологии.  
Максимальная простота и дружелюбие.  
Никаких перегруженных B2B-интерфейсов.

---

## 3. Structure (Initial Phase) / Структура (первая версия)

**EN:**  
Initial version consists of 3–4 logical sections/pages:
- Home
- Projects
- About
- Contact / Access

Login access must be visible from the main page.

**RU:**  
Первая версия сайта состоит из 3–4 логических разделов:
- Главная
- Проекты
- О платформе
- Контакты / Вход

Вход в личный кабинет должен быть доступен с главной страницы.

---

## 4. Projects Integration / Интеграция проектов

**EN:**  
The homepage presents LabsBot projects as independent modules/cards.

Initial projects:
- MigrantBot (active)
- PayTable (in development)
- Other bots/services (placeholders allowed)

Each project:
- has its own identity
- may become a standalone web application
- is accessible via LabsBot.ai

LabsBot.ai remains a hub, not a monolith.

**RU:**  
На главной странице проекты LabsBot отображаются как отдельные модули / карточки.

Начальные проекты:
- MigrantBot (рабочий)
- PayTable (в разработке)
- Другие сервисы (пока заглушки)

Каждый проект:
- имеет собственную идентичность
- может стать отдельным веб-приложением
- доступен через LabsBot.ai

LabsBot.ai — это хаб, а не монолит.

---

## 5. Technical Philosophy / Техническая философия

**EN:**  
- Minimal initial implementation
- Clean structure over visual effects
- No tight coupling between site and project internals
- Easy to extend
- Easy to redesign

**RU:**  
- Минимальная стартовая реализация
- Чистая структура важнее визуальных эффектов
- Нет жёсткой связки с внутренностями проектов
- Простое расширение
- Возможность редизайна без переписывания всего

---

## 6. Styling & UI / Стиль и интерфейс

**EN:**  
At start:
- Plain HTML structure is acceptable
- Focus on semantics and layout

Later:
- Utility-first styling (e.g. Tailwind) MAY be added
- Styling must not block future redesign

Design tone:
- modern
- calm
- trustworthy
- not flashy
- not aggressive marketing

**RU:**  
На старте:
- Допустим простой HTML
- Акцент на структуру и смысл

Позже:
- Возможна утилитарная стилизация (например Tailwind)
- Ничто не должно мешать будущему редизайну

Тон дизайна:
- современный
- спокойный
- вызывающий доверие
- без маркетинговой агрессии

---

## 7. Deployment / Деплой

**EN:**  
Deployment is handled via:
- GitHub
- DigitalOcean App Platform
- Docker

Deployment pipeline must not be changed without explicit decision.

**RU:**  
Деплой осуществляется через:
- GitHub
- DigitalOcean App Platform
- Docker

Пайплайн деплоя не трогается без отдельного решения.

---

## 8. Non-Goals / Чего это НЕ

**EN:**  
- Not a one-page promo
- Not a throwaway prototype
- Not a temporary mockup

**RU:**  
- Не одностраничный промо-сайт
- Не одноразовый прототип
- Не временная заглушка

---

**EN:**  
LabsBot.ai is a long-term platform.

**RU:**  
LabsBot.ai — это долгосрочный проект.
